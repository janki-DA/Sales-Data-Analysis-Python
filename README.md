# Sales Data Analysis using Python

## Project Overview
This project analyzes a sales dataset of 1000 records using Python. 
The goal is to clean the data, explore patterns, and visualize sales performance.

## Tools & Libraries Used
- Python
- NumPy
- Pandas
- Matplotlib

## Dataset
The dataset contains 1000 sales records with the following columns:
- Region
- Country
- Item Type
- Sales Channel
- Order Priority
- Order Date
- Order ID
- Ship Date
- Units Sold
- Unit Price
- Unit Cost
- Total Revenue
- Total Cost
- Total Profit

## Steps Performed
1. Data Loading
2. Data Cleaning
3. Date Conversion
4. Conditional Filtering
5. Exploratory Data Analysis
6. Data Visualization

## Visualizations Created
- Bar Chart (Revenue by Region)
- Bar Chart (Profit by Category)
- Pie Chart (Revenue Distribution)
- Line Chart (Yearly Revenue Trend)
- Histogram (Delivery Days Distribution)

## Key Insights
- Identified the highest revenue generating region
- Compared profit across product categories
- Observed yearly revenue growth pattern
- Analyzed delivery time distribution

## Conclusion
This project demonstrates how Python can be used for data cleaning, analysis, and visualization. It helps in understanding sales trends and business performance using real-world data.
